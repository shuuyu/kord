A representation of a Discord Channel Structure.

Parameters

id

The id of the channel.

type

the Type of channel.

guildId

the id of the guild.

position

The sorting position of the channel.

permissionOverwrites

The explicit permission overwrite for members and roles.

name

The name of the channel.

topic

The channel topic.

nsfw

Whether the channel is nsfw.

lastMessageId

The id of the last message sent in this channel (may not point to an existing or valid message).

bitrate

The bitrate (in bits) of the voice channel.

userLimit

The user limit of the voice channel.

rateLimitPerUser

amount of time a user has to wait before sending another message; bots, as well as users with the permission Permission.ManageMessages or Permission.ManageChannels are unaffected.

recipients

The recipients of the DM.

icon

The icon hash.

ownerId

The id of DM creator.

applicationId

The application id of the group DM creator if it is bot-created.

parentId

The id of the parent category for a channel.

lastPinTimestamp

When the last pinned message was pinned.
